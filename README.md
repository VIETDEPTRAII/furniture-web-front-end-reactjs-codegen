# Furniture Web App - React Frontend

A modern, responsive React application for a furniture e-commerce website.

## Features

- Responsive design for all devices
- Modern UI with styled-components
- React Router for navigation
- Landing page with multiple sections:
  - Hero section
  - Featured Products
  - Categories
  - Testimonials
  - Newsletter signup

## Getting Started

### Prerequisites

- Node.js (v14 or later)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/VIETDEPTRAII/furniture-web-front-end-reactjs-codegen.git
cd furniture-web-front-end-reactjs-codegen
```

2. Install dependencies:
```bash
npm install
# or
yarn install
```

3. Start the development server:
```bash
npm start
# or
yarn start
```

4. Open your browser and navigate to `http://localhost:3000`
